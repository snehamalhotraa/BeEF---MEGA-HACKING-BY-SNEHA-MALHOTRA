# Mega Hacking Project – BeEF Lab (Sneha Malhotra)

This repo contains my Browser Exploitation Framework (BeEF) lab for "THE MEGA HACKING MEGA GROUP PROJECT OF DEATH" assignment.
My partner completed the Juice Shop part. I completed the BeEF / client-side exploitation exercise. 

# 1. Lab Overview
Attacker VM: Kali Linux with BeEF and Apache2
Victim VM: "Victim Machine" (Linux with Firefox)
Network: Host-only / NAT, no internet targets
Goal: Show how a browser can be controlled after visiting a page that loads the BeEF hook script, and then describe how to block this.

All details, screenshots, and explanations are inside:
 `Mega hacking.pdf`
